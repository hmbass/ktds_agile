# kt ds Agile Playbook

### 개요

kt ds Agile Playbook은 실무에 바로 적용할 수 있는 가이드라인으로서 kt ds에 특화된 방법론입니다. 각 단계별 Activity 에 주요 설명 및 기법들이 기술되어 있으며 순서대로 따라하기 방식으로 구성되어 있습니다.
